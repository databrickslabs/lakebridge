See [https://docs.snowflake.com/en/sql-reference/snowflake-scripting/if](https://docs.snowflake.com/en/sql-reference/snowflake-scripting/if)
```
IF ( <condition> ) THEN
    <statement>;
    [ <statement>; ... ]
[
ELSEIF ( <condition> ) THEN
    <statement>;
    [ <statement>; ... ]
]
[
ELSE
    <statement>;
    [ <statement>; ... ]
]
END IF;
```
